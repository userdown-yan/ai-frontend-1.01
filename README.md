# Growth OS - AI 对话与个人成长中枢

基于 Vue3 + Spring Boot 开发的全栈项目，支持账号体系、AI 对话、个人收藏与任务管理。

---

## 📸 项目截图展示

### 功能界面 1：AI 对话中枢（核心模块）
![功能界面1](1.png)
- **多模型对话交互**：支持选择不同的 AI 模型（如界面中的 `gpt-4o-mini`），用户可通过输入框发送文本消息，实现与 AI 的实时对话；支持`Enter发送消息`、`Shift+Enter换行`的快捷输入方式。
- **会话管理系统**：支持创建新对话、自动保存会话历史，提供会话搜索、列表展示与切换功能，方便管理多轮对话场景。
- **模板化对话支持**：内置对话模板和应用模板，可快速加载预设场景或配置，提升对话效率。
- **多模态上传能力**：支持图片等非文本内容上传，实现图文结合的多模态交互。
- **界面与交互设计**：左侧为会话管理面板，右侧为对话交互主面板；侧边导航栏提供`AI Chat`/`Dashboard`/`Tasks`模块快速切换入口；右上角支持用户登出，整体交互逻辑贴合主流AI对话产品使用习惯。

### 功能界面 2：Dashboard 数据总览与个人中心
![功能界面2](2.png)
- **数据可视化概览**：展示总对话数、收藏数、使用天数等核心使用数据，直观呈现个人成长与使用概况。
- **最近内容展示**：清晰展示最近收藏的笔记、对话记录及历史对话，支持快速查看与回顾。
- **个人中心功能**：左侧提供个人信息、我的收藏、历史对话、账号设置、主题切换等完整个人管理入口。
- **快捷操作按钮**：支持一键新建对话、查看收藏、生成报告，提升操作效率。
- **系统模块切换**：可快速跳转 AI 对话、任务管理、数据面板等核心功能模块。

### 功能界面 3：我的收藏管理
![功能界面3](3.png)
- **收藏内容管理**：支持对收藏的对话进行统一管理，每条收藏记录显示标题、简介、创建时间与分类标签（如产品、分析、前端等）。
- **多维度筛选**：提供实时搜索、分类筛选、标签筛选与分页功能，可快速定位目标收藏内容。
- **快捷操作功能**：支持「跳转对话」快速回溯原对话场景、「复制内容」一键提取对话文本、「取消收藏」便捷移除记录。
- **个人中心入口**：左侧导航栏提供个人信息、历史对话、账号设置等功能入口，方便一站式管理个人数据。
- **模块化切换**：可快速切换 AI Chat、Dashboard、Tasks 等核心功能模块，保持操作连贯性。

### 功能界面 4：历史对话归档管理
![功能界面4](4.png)
- **对话归档管理**：对所有历史对话按日期自动归档存储，支持长期留存与追溯对话内容。
- **快速检索功能**：提供实时搜索框，可通过关键词快速定位目标历史对话。
- **继续对话支持**：可直接从历史记录中恢复对话上下文，实现无缝续聊。
- **空状态友好提示**：暂无对话记录时展示清晰的占位提示，引导用户开始使用。
- **个人中心导航**：左侧菜单提供总览首页、我的收藏、账号设置等入口，支持一站式个人数据管理。

### 功能界面 5：个人中心与主题适配
![功能界面5](5.png)
- **数据总览首页**：集中展示用户核心使用数据，包括总对话数、收藏数、使用天数等关键指标，直观呈现使用概况。
- **内容快捷访问**：展示最近收藏的对话内容与最近对话记录，支持快速查看与回顾。
- **主题切换功能**：支持一键切换系统主题，切换后实时提示“主题切换成功”，适配不同使用场景与视觉偏好。
- **快捷操作入口**：提供「新建对话」「查看收藏」「生成报告」等快捷按钮，支持一键直达核心功能，提升使用效率。
- **个人管理导航**：左侧菜单提供个人信息、账号设置、主题切换等完整个人中心功能入口，支持一站式账号与数据管理。

## 项目介绍
- 前端：Vue3 + Vite + Element Plus
- 后端：Spring Boot + MyBatis-Plus + MySQL
- 核心功能：AI 对话、对话收藏、用户中心、任务管理
