# Growth OS - AI 对话与个人成长中枢

基于 Vue3 + Spring Boot 开发的全栈项目，支持账号体系、AI 对话、个人收藏与任务管理。

---

## 📸 项目截图展示

### 功能界面 1：AI 对话中枢（核心模块）
![功能界面1](1.png)
- **多模型对话交互**：支持选择不同的 AI 模型（如界面中的 `gpt-4o-mini`），用户可通过输入框发送文本消息，实现与 AI 的实时对话；支持`Enter发送消息`、`Shift+Enter换行`的快捷输入方式。
- **会话管理系统**：支持创建新对话、自动保存会话历史，提供会话搜索、列表展示与切换功能，方便管理多轮对话场景。
- **模板化对话支持**：内置对话模板和应用模板，可快速加载预设场景或配置，提升对话效率。
- **多模态上传能力**：支持图片等非文本内容上传，实现图文结合的多模态交互。
- **界面与交互设计**：左侧为会话管理面板，右侧为对话交互主面板；侧边导航栏提供`AI Chat`/`Dashboard`/`Tasks`模块快速切换入口；右上角支持用户登出，整体交互逻辑贴合主流AI对话产品使用习惯。

## 项目介绍
- 前端：Vue3 + Vite + Element Plus
- 后端：Spring Boot + MyBatis-Plus + MySQL
- 核心功能：AI 对话、对话收藏、用户中心、任务管理
